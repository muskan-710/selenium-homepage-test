# Selenium Homepage Test

## Project Overview
This project automates homepage testing using Selenium WebDriver and Python.

## Features
- Opens the target website
- Validates homepage elements
- Scrolls through the page
- Captures screenshot
- Automated browser testing

## Technologies Used
- Python
- Selenium WebDriver
- ChromeDriver
- Git
- GitHub

## Project Structure
- test_homepage.py
- main.py
- homepage.png
